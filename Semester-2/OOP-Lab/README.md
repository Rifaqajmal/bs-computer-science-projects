# Object Oriented Programming Lab – Semester 2

This folder contains Object Oriented Programming (OOP) practice programs developed during my second semester of BS Computer Science.

## Concepts Covered

- Classes and Objects
- Constructors
- Function Overloading
- Operator Overloading
- Inheritance
- Multilevel Inheritance
- Polymorphism
- Encapsulation
- File Handling
