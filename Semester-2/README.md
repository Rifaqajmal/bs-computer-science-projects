### 2. Digital Logic Design (DLD)

This folder contains the semester project developed for the Digital Logic Design course.

The project was designed using **Multisim** and demonstrates how digital logic components can be used to build a simple password verification system.

Main Components Used:

- 74LS85 (4-bit magnitude comparator)
- D Flip-Flops
- Logic gates

Files included:

- Multisim project file (.ms14)
- Project report (.pdf)
