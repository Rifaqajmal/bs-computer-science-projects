# Database Management Systems (DBMS) – SQL Lab Work

This folder contains SQL practice programs and lab work completed during the **Database Management Systems (DBMS)** course in the **third semester of my BS Computer Science program**.

The SQL scripts demonstrate practical database operations such as table creation, data manipulation, data retrieval, and query optimization.

---

## Topics Covered

The lab tasks in this folder cover several core database concepts, including:

- Table creation and constraints
- Data insertion using INSERT
- Data retrieval using SELECT
- Update and delete operations
- Filtering with WHERE conditions
- Sorting and limiting results
- Aggregate functions
- Grouping data with GROUP BY and HAVING
- Join operations
- Subqueries
- Views
- Indexing

---

## Files Included

The SQL scripts are organized sequentially to follow the learning progression:

- 01 – Create Student Table
- 02 – Insert Student Data
- 03 – Select Queries
- 04 – Update and Delete Operations
- 05 – WHERE Conditions
- 06 – ORDER BY and LIMIT
- 07 – Aggregate Functions
- 08 – GROUP BY and HAVING
- 09 – Join Queries
- 10 – Subqueries
- 11 – Views
- 12 – Indexes

Each file demonstrates a specific SQL concept used in relational database management systems.

---

## How to Run

These SQL scripts can be executed using any relational database system such as:

- MySQL
- PostgreSQL
- SQLite

Run the files sequentially starting from:

01_create_student_table.sql

and continue in order to understand the complete workflow.
