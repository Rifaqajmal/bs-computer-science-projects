# Computer Organization & Assembly Language (COAL) – Lab Work

This folder contains assembly language programs and lab work completed during the **Computer Organization and Assembly Language (COAL)** course in the **third semester of my BS Computer Science program**.

All programs were written and tested using the **EMU8086 emulator**, focusing on fundamental concepts of the **8086 microprocessor architecture** and low-level programming.

---

## Topics Covered

The lab exercises demonstrate several important assembly language concepts, including:

- Basic program structure
- Input and output operations
- Data types and memory addressing
- Jump instructions and loop structures
- Arrays and string processing
- Stack operations
- Procedures and macros
- Multiplication and division instructions
- Shift and rotate operations
- Conditional logic and program flow control

---
