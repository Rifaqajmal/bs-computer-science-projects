# Semester 3 – BS Computer Science

This folder contains coursework and lab work completed during the **third semester of my Bachelor of Science in Computer Science program**.

The repositories included here demonstrate practical work in **data structures, database systems, and computer organization**, implemented through programming assignments, lab exercises, and reports.

---

## Subjects Included

### Data Structures and Algorithms (DSA)

This folder contains C++ implementations of fundamental data structures and algorithms.  
The programs cover topics such as arrays, stacks, queues, linked lists, trees, sorting algorithms, and basic algorithm analysis.

Key Concepts:
- Arrays and pointers
- Stack, Queue, and Deque
- Sorting algorithms
- Linked lists and circular linked lists
- Binary Search Trees (BST)
- Basic algorithm analysis

---

### Computer Organization and Assembly Language (COAL)

This folder contains assembly language programs developed using the **EMU8086 emulator**.  
The labs demonstrate core concepts of the **8086 microprocessor architecture**, instruction sets, and low-level programming.

Key Concepts:
- Assembly program structure
- Interrupt handling
- Loops and conditional jumps
- Stack operations
- Procedures and macros
- Shift and rotate instructions

---

### Database Management Systems (DBMS)

This folder contains SQL scripts demonstrating fundamental database operations and query techniques used in relational database systems.

Key Concepts:
- Table creation and constraints
- Data insertion and retrieval
- Filtering and sorting data
- Aggregate functions
- Joins and subqueries
- Views and indexing

The purpose of this repository is to **organize my semester coursework, document programming practice, and build a structured academic portfolio on GitHub**.
