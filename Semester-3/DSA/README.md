# Data Structures & Algorithms (DSA) – Lab Work

This folder contains C++ programs and lab work completed during the **Data Structures and Algorithms (DSA)** course in the **third semester of my BS Computer Science program**.

The implementations focus on understanding fundamental data structures, algorithm design, and basic performance analysis through practical programming exercises.

---

## Topics Covered

The lab tasks in this folder demonstrate several important concepts in Data Structures and Algorithms, including:

- Arrays and pointer operations
- Object-oriented programming concepts
- Stack, Queue, and Deque implementations
- Sorting algorithms
- Singly linked lists
- Circular linked lists
- Linked list sorting and time analysis
- Binary Search Trees (BST)
- Basic data compression algorithms

---

## Lab Programs

The programs are organized according to different lab exercises:

- **Lab 1–2:** Arrays, Pointers, and Class Basics  
- **Lab 3:** Stack, Queue, and Deque  
- **Lab 4:** Sorting Algorithms (Bubble Sort, Insertion Sort)  
- **Lab 5:** Singly Linked List Operations  
- **Lab 6:** Circular Linked List Operations  
- **Lab 7:** Linked List Sorting and Execution Time Analysis  
- **Lab 8:** Binary Search Tree (BST) Implementation

## Open Ended Lab (OEL)

**Run-Length Encoding (RLE)**

This implementation demonstrates a simple **data compression technique** that reduces repeated characters in a string.

Example:

Input: `AAABBBCCDAA`  
Output: `A3B3C2D1A2`
