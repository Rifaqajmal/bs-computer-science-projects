# Artificial Intelligence – Lab Work

This folder contains laboratory work completed during the **Artificial Intelligence course** in the **fourth semester** of the BS Computer Science program.

The labs include Python programs, datasets, Jupyter notebooks, and reports that demonstrate practical concepts used in introductory AI and data processing tasks.

---

## Contents

* **Lab Folders (Lab-01 to Lab-05)**
  Python implementations along with datasets and lab reports.

* **Jupyter Notebooks (Lab-06 to Lab-13)**
  Experiments and implementations performed using **Jupyter Notebook**.

* **Lab-14**
  Python implementation with dataset and supporting files.

* **Semester Project**
  A desktop application (**Nexus File Organizer**) developed in Python that organizes files automatically based on type, size, date, or alphabetical order.

---

## Technologies Used

* **Python**
* **Jupyter Notebook**
* **Pandas**
* **Matplotlib**
* **CustomTkinter**
* **Excel datasets**

---

## Purpose

The objective of these labs was to understand practical applications of **Artificial Intelligence concepts, data handling, and algorithmic problem solving using Python**.
