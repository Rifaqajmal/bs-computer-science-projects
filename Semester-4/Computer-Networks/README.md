# Computer Networks – Lab Work

This folder contains lab reports and network simulation files completed during the **Computer Networks course** in the **fourth semester of my BS Computer Science program**.

The repository includes practical networking exercises, packet analysis, and network configuration tasks performed using simulation and analysis tools.

---

## Contents

- **Lab Reports (PDF):** Documentation of lab tasks, configurations, and observations.
- **Cisco Packet Tracer Files (.pkt):** Network topologies and configurations created during the labs.

---

## Tools & Technologies

- **Cisco Packet Tracer** – Network design and simulation  
- **Wireshark** – Packet capture and traffic analysis  
- **Python (Requests Library)** – Basic HTTP request experiments  
- **Command Line Tools** – `ipconfig`, `ping`, DNS lookup

---

## Lab Topics

The labs cover several networking concepts, including:

- Basic networking configuration and troubleshooting  
- Packet analysis using Wireshark  
- HTTP and DNS traffic analysis  
- DHCP, DNS, and Web server configuration  
- Switch-based and router-based network setups  
- Routing protocols such as **RIP** and **OSPF**
